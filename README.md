
## 🔧 Технические детали

- **Язык:** C#
- **Тип приложения:** Консольное
- **Алгоритмы:** Вагнера-Фишера для расстояния Левенштейна и Дамерау-Левенштейна
- **Сложность:** O(n × m), где n и m — длины строк

## 📁 Файлы

- `project.cs` — основной код программы

## 🎓 Теоретическая справка

### Расстояние Левенштейна
Минимальное количество операций:
- Удаление символа
- Вставка символа
- Замена символа

### Расстояние Дамерау-Левенштейна
Добавляет четвёртую операцию:
- Перестановка двух соседних символов (транспозиция)
